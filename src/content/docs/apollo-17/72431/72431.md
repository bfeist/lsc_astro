# **72431**

## Soil 79.5 grams

![](_page_0_Figure_2.jpeg)

*Figure 1: Map for station 2, Apollo 17.* 

### **Introduction**

72430 is the soil scooped up when 72435 was picked up from the regolith after it was chipped from the boulder and it was returned in the same bag.. It has exactly the same composition as the regolith from beneath the boulder (74240, 74260).

### **Petrography**

Morris (1978) determined the maturity index (Is /FeO =63). Meyer (1973) cataloged the coarse-fines.

### **Chemistry**

Korotev and Kremser (1992) determined the chemical composition by neutron activation analysis (table).
